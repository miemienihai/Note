## @interface 
+ 源码中看到这个代表他是一个注解类.


## @Target(ElementType.METTHD) 元注解
+ Target代表这个注解你可以放在那些元素上,这个表明我们可以放在方法上.
+ 元注解修饰注解的注解
  
## Retention
+ 注解作用的范围 三种SOURCE CLASS ,RUNTIME
  

## Documented 
+ 指定该注解是否出现在javadoc体现

## Inherited
+ 子类会继承父类注解
